XHTML Validation

$Provider supports web standards. All of the pages across the $Provider website have been created with a strict adherence to the XHTML specification. The W3C is responsible for creating and maintaining the XHTML standard. Along with the specification the W3C provides set of DTD that developers can use to verify that their pages precisely meet the W3C standards.

Here are links you can be use to validate all of the $Provider web pages against the XHTML Strict DTD. $Provider is proud of its support for web standards and encourages developers to validate their web pages against this specification.
