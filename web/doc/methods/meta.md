Displays meta info in the main page. Browser, cookies, stylesheets, and connection security are detected by client.

### Response Params

"userID"

- integer

"clientIP"

- string

"location"

- string

"reputation" - *0 to 100*

- integer

"plan"

1. "Free"
1. "Basic"
1. "Personal"
1. "Enhanced"
1. "Premium"

"version" - *webmail version*

- string

"quota" - *0 to 100*

- integer
