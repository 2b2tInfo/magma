Assign various flags to a message. The flag property indicates what flag is being assigned.

### Request Params

Each object in the example is a separate request.

"action"

- string
 1. add
 1. replace
 1. remove

"flags" 

- array of strings
 - read
 - unread
 - star
 - unstar

"messageIDs"

- array

"folderID"

- integer
